# ComToAir

Устройство для приема данных по интерфейсу RS-232 и трансляции их через WiFi в виде веб-сервиса на базе Seeed Studio XIAO ESP32-C6.

## Описание

ComToAir - это устройство, которое принимает данные от устройств с интерфейсом RS-232 и предоставляет доступ к этим данным через веб-интерфейс по WiFi. Устройство построено на базе микроконтроллера ESP32-C6 от Seeed Studio.

## Основные возможности

- Прием данных по RS-232 с настраиваемыми параметрами
- Подключение к WiFi сети
- Веб-сервер для доступа к данным
- RESTful API
- Веб-интерфейс для мониторинга в реальном времени
- Настройка через веб-интерфейс

## Требования

### Аппаратное обеспечение
- Seeed Studio XIAO ESP32-C6
- Преобразователь уровня TTL ↔ RS-232 (например, MAX232)
- Источник питания 5V

### Программное обеспечение
- Visual Studio Code
- PlatformIO IDE extension
- ESP-IDF framework

## Установка

1. Установите Visual Studio Code
2. Установите расширение PlatformIO IDE
3. Клонируйте или скопируйте проект в рабочую директорию
4. Откройте проект в PlatformIO
5. Подключите устройство и загрузите прошивку

## Использование

1. Подключите устройство с RS-232 к преобразователю уровня
2. Подключите преобразователь к XIAO ESP32-C6
3. Настройте WiFi через веб-интерфейс (при первом запуске устройство создаст точку доступа)
4. Откройте веб-интерфейс в браузере
5. Настройте параметры RS-232
6. Начните мониторинг данных

## API

- `GET /` - главная страница
- `GET /api/data` - получение последних данных
- `GET /api/status` - статус устройства
- `GET /api/config` - текущая конфигурация
- `POST /api/config` - изменение конфигурации
